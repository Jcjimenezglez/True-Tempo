# SEO Setup Guide - Superfocus

Domain: https://www.superfocus.live

## Completed SEO Optimizations

### Meta Tags & HTML Structure
- Title, description, keywords per page (homepage + ~100 pSEO landing pages)
- Open Graph + Twitter Cards (`og:site_name`, `twitter:site`)
- Canonical URLs on all indexable pages
- Google Search Console verification placeholder in `index.html`

### Structured Data (JSON-LD)
- Homepage: Organization, SoftwareApplication, HowTo, FAQPage
- pSEO pages: dynamic WebPage, BreadcrumbList, category-specific FAQPage / HowTo / SoftwareApplication
- Pricing: SoftwareApplication with Offer

### Technical SEO Files
- `sitemap.xml` — auto-regenerated by `npm run build:pseo` (includes `/contact`)
- `robots.txt` — allows crawlers, blocks `/api/`
- `llms.txt` — AI crawler summary at site root
- `.well-known/security.txt` — security contact

### Performance
- pSEO pages use lightweight `pseo/template.html` (no full app shell)
- `npm run build:assets` — esbuild minify to `/dist/` with immutable cache headers
- Deferred analytics via `analytics-deferred.js`
- Core Web Vitals reporting via web-vitals → GA4

## Build Commands

```bash
npm run build          # apply pain copy + minify assets + regenerate pSEO pages
npm run build:assets   # minify JS/CSS to dist/
npm run build:pseo     # regenerate landing pages + sitemap
npm run apply-pain-copy # update copy fields in pseo JSON sources
```

## Required Manual Setup

### Google Search Console
1. Go to [Google Search Console](https://search.google.com/search-console/)
2. Add property: `https://www.superfocus.live`
3. Copy verification meta tag into `index.html` (replace placeholder comment)
4. Submit sitemap: `https://www.superfocus.live/sitemap.xml`

### Google Analytics
- Measurement ID `G-T3T0PES8C0` configured (loaded via deferred analytics)

### OG Image
- Generate with `npm run update-logo` or ensure `og-image.png` exists at site root (1200×630)

## Monitoring

- PageSpeed Insights: test homepage + 3 pSEO URLs after deploy
- GSC: monitor impressions on `/use-cases/` and `/compare/` paths
- Mixpanel: track `ref=pseo-{slug}` conversion from landing CTAs
